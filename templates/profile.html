FacePass

User Profile

Represents a user with a unique identity within the system.

  • ID: A unique identifier for the user profile.
  • Name: The name of the user.
  • Access Permissions: A list of permissions associated with the user.

Access Permission

Specifies the level of access granted to a user profile.

  • ID: A unique identifier for the access permission record.
  • User Profile ID: The identifier for the related user profile.
  • Permission Level: The specific access level granted.

Access Request

Logs an access attempt by a user, storing the outcome and any associated permissions.

  • ID: A unique identifier for the access request.
  • Timestamp: The date and time when the access was requested.
  • Outcome: The result of the access attempt.
  • Associated Permission: The level of access requested.

Access Log

Records details about access attempts and outcomes for audit and monitoring purposes.

  • ID: A unique identifier for the log entry.
  • Access Request ID: Links to the related access request.
  • Details: Additional information about the access event.
